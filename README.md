# jameslockman/homebrew-tap

Homebrew tap for [Griffin PowerMate Driver](https://github.com/jameslockman/Griffin-PowerMate-Driver).

## Install

```bash
brew tap jameslockman/tap
brew install --cask griffin-powermate-driver
```

Or in a single command:

```bash
brew install --cask jameslockman/tap/griffin-powermate-driver
```

## Upgrade

```bash
brew upgrade --cask griffin-powermate-driver
```

## Uninstall

```bash
brew uninstall --cask griffin-powermate-driver
```

To also remove all preferences and support files:

```bash
brew uninstall --zap --cask griffin-powermate-driver
```
